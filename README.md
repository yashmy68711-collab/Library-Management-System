# Python Library Management System

## About

A simple CLI Library Management System built using Python.
This application helps users manage books by viewing available books, borrowing books, returning books, and adding new books to the library.

---

## ✨ Features

✔️ View available books
✔️ Borrow books
✔️ Return books
✔️ Add new books
✔️ Transaction history
✔️ Duplicate book prevention
✔️ Smart case-insensitive search

---

## 🛠️ Tech Used

* Python
* Object-Oriented Programming (OOP)

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Choose options from the menu
5. Manage library books
